# Image resolutions

eZ Commerce delivers the following image resolutions:

| Name                          | Resolution (in px) |
| ----------------------------- | ------------- |
| `thumb_smallest`               | 81x61         |
| `thumb_smaller`                | 100x75        |
| `thumb_small`                  | 150x113       |
| `thumb_medium`                 | 420x315       |
| `thumb_big`                    | 420x600       |
| `image_zoom`                   | 1600x1200     |
| `slick_tabs_homepage_slider` | 420x315       |
