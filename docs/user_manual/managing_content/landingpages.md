<p>This feature can be used e.g. to display a list of blog posts or events.</p>

<p><strong>Important</strong>: after choosing the location you need to set the ContentTypes which shall be displayed. Example: for Blogpost you need to select "Blog post" and for Events "event".</p>
<p>If you choose a wrong Contenttype an empty list might be displayed.</p>

<td><p>With the gallery block (customized for the shop) its possible to add a slider with several images on a landingpage:</p>
<ol>
<li>Before you start you have to upload your images to a folder in the Media Libary</li>
<li>With the prepared Folder you can go to the Page Tab of the Landingpage you want to edit</li>
<li>Select the desired Folder with the images for the Slider</li>
<li>When click on Submit you will see the First image of the Slider in the preview, now you are ready to publish the landingpage</li>
</ol></td>
